SKILLS

  • HTML

    웹 표준과 웹 접근성을 준수하는 HTML 정보
    구조 설계로 웹사이트 퍼블리싱을 하고 있습니다.
    HTML5에서 권장하는 마크업 방식과 시멘틱
    태그를 사용하여 전체적인 레이아웃을
    설계합니다. 또한 웹사이트를 퍼블리싱 하기 전
    기능정의서를 만들어 체계적인 퍼블리싱
    작업을 하고자 노력합니다.

  • CSS

    크로스브라우징을 위해 안정적인 CSS 속성을
    사용하여 HTML Elements를 배치합니다.
    CSS3에서 새롭게 지원하는 속성을 활용하여
    다양한 CSS배치 및 디자인을 할 수 있습니다.

  • JS

    네비게이션, 탭 메뉴, 아코디언 등 웹 퍼블리싱에
    반드시 필요한 상호작용은 플러그인을
    사용하지 않고 순수 제이쿼리 코딩으로 제작합니다.
    필수적인 자바스크립트 활용이 가능하며
    제이쿼리와 병행하여 사용합니다.

  • RESPONSIVE WEB

    크로스브라우징을 위해 안정적인 기존 방식
    float, position, media Query를 활용한
    웹 제작 경험을 충분히 가지고 있습니다.
    모바일 웹 앱 등 필요에 따라 flex, grid
    방식으로 반응형 웹 사이트를 제작합니다.

  • MOBILE WEB APP

    반응형 웹과 함께 모바일 대응 플랫폼에 대해
    이해하고 있으며, Mobile First(모바일 우선)로
    제작할 수 있습니다. UI UX 측면에서
    미적인 디자인 뿐 아니라 모바일 사용자 경험에
    최적화를 위한 효율적인 컨텐츠 배치를
    우선으로 생각하며 제작합니다.

  • GIT

    효율적인 버전관리 및 협업을 위해 git을
    사용할 수 있습니다.

  • VS CODE

    텍스트 에디터는 Visual Studio Code를
    메인으로 사용합니다.

  • PHOTOSHOP / ILLUSTRATOR

    Adobe Photoshop, Adobe Illustrator를
    활용하여 웹사이트 UI 디자인
    작업 및 협업을 할 수 있습니다.

  • XD

    Adobe XD를 활용하여 웹사이트 UI 디자인,
    프로토타입 작업 및 협업을 할 수 있습니다.
